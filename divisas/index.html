API Pública (Para Desarrolladores)

Nuestra sección de divisas funciona gracias a nuestra propia API Pública, propulsada por Netlify Functions y datos del BCE. Puedes consumirla de manera gratuita, sin registrarte y sin keys directamente desde nuestra web.

URL Base

https://dolarctd.fabrictd.com/api/v2

Endpoints Principales

Obtener todas las divisas disponibles:

GET /v2/currencies

Obtener el tipo de cambio actual (Último disponible):

GET /v2/latest?base=USD&symbols=EUR,GBP

Obtener datos históricos (Ej: de 2020 a 2023):

GET /v2/rates?base=USD"es=EUR,GBP&from=2020-01-01&to=2023-12-31
